# Android Source Structure

- app/: Android entry points (Activity/Application)
- data/model/: data models (entities/form state)
- data/repository/: repository and data providers
- ui/pages/: screen-level composables
- ui/components/: reusable UI components
- ui/hooks/: UI-related state helpers (future)
- ui/services/: UI-facing service adapters (future)
- ui/store/: app state containers (future)
- ui/types/: UI-specific types (future)
- ui/utils/: UI utilities (future)
- ui/theme/: Compose theme system
